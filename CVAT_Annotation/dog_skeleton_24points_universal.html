üí° Why 24 Points?

  • SLEAP Compatibility: Provides detailed skeleton for research-grade tracking
  • Anatomically Correct: Uses proper veterinary terminology (hip, knee, not "back_shoulder")
  • Annotation Efficiency: Can annotate core 17 points first, add details later
  • Stanford Dataset Aligned: Compatible with existing 20-24 point dog datasets
